# VolunThings — Frontend

Frontend de la aplicación **VolunThings**, desarrollado con **React + Vite**, es una aplicación diseñada para el control de material que una ONG presta o usa internamente.

## Comprobaciones previas antes de arrancar el proyecto
- Asegúrate de tener instalado [Node.js](https://nodejs.org/)

- Debes de tener la **API de VolunThings** ejecutándose para que pueda conectarse correctamente.

---

## Instalación y ejecución

1. Es necesario que instales las dependencias:
```bash
npm install
```

2. Ejecutas la aplicación en modo desarrollo:
```bash
npm run dev
```

3. Al ejecutarse la aplicación, puedes pegar la siguiente URL en tu navegador web:

La aplicación estará disponible en `http://localhost:5173` por defecto.
